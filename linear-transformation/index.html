3x3行列で学ぶ 2D 座標変換（D3.js）

同次座標(3x3行列)で一次変換・平行移動・回転を体感しながら学べるミニサイト

レッスン一覧

  1. 一次変換（拡大縮小・せん断・反転の合成）
  2. 平行移動
  3. 一次変換 + 平行移動（アフィン変換）
  4. 原点中心の回転
  5. 任意の点中心の回転（T(-p)・R・T(p) を表示）

使い方

  • 右上のスライダーや数値入力でパラメータを変えると、図形や基底ベクトルが変換されます。
  • 表には実際に使用されている 3x3 行列が表示されます（同次座標）。
  • 原点中心以外の回転は T(p) と T(-p)、R(θ) の 3行列の積で実装・表示しています。
